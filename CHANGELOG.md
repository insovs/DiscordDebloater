## [1.0] — 2025-06-12

### Added
- Initial release
- Debloat section: removes old app versions, unused native modules, language packs, junk files
- Optimize Settings section: writes a performance-focused `settings.json`
- Clean Cache section: clears all major Discord AppData cache folders
- Advanced options: backup, Krisp removal, Game SDK, RPC, auto-updater, autostart, FSO
- Auto-creates `Discord Debloated.lnk` shortcut on Desktop after debloating
- GUI built with PowerShell WPF/XAML — no install required
